This project contains 10 coding tasks, each targeting a distinct capability category. Every task is self-contained with all necessary code, specs, and evaluation criteria embedded in the task description.

## Categories

1. Code Generation -- write a module from a spec
2. Bug Fixing -- find and fix a bug in provided code
3. Refactoring -- improve code structure without changing behavior
4. Code Review -- review code and provide actionable feedback
5. Test Writing -- write comprehensive tests for existing code
6. Documentation -- generate docs from code
7. Multi-file Changes -- coordinated edits across multiple files
8. Debugging -- diagnose an issue from error output
9. Architecture -- design a system from requirements
10. Performance Optimization -- improve code performance

## Evaluation

Each task defines its own evaluation criteria with weighted scoring. The Test Director scores results 0-5 using the standard rubric.
